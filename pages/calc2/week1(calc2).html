Interactive Calculus: Area Under Curve

Problem Statement

Find the area of the region that lies to the right of the y-axis and to the left of the parabola $$x = 2y - y^2$$ from $$y = 0$$ to $$y = 2$$.

Step 1: Set up the integral

\[\int_0^2 (2y - y^2) dy\]
We set up the integral based on the area we want to calculate. The limits are from y=0 to y=2, and the integrand is the function describing the curve.

Step 2: Integrate

\[\int_0^2 (2y - y^2) dy = \left[y^2 - \frac{y^3}{3}\right]_0^2\]
We integrate each term separately. For 2y, we get y^2. For -y^2, we get -y^3/3.

Step 3: Evaluate the integral

\[\left(2^2 - \frac{2^3}{3}\right) - \left(0^2 - \frac{0^3}{3}\right) = 4 - \frac{8}{3} = \frac{4}{3}\]
We substitute the upper and lower limits and calculate the difference.
The area of the region is $$\frac{4}{3}$$ square units.

Practice Problems

Problem 1

\[\int (x^{1.3} + 7x^{2.5}) dx\]

Step 1: Split the integral

\[\int x^{1.3} dx + \int 7x^{2.5} dx\]
We split the integral using the sum rule of integration. This allows us to integrate each term separately.

Step 2: Integrate x^1.3

\[\int x^{1.3} dx = \frac{x^{2.3}}{2.3} + C\]
We use the power rule: $$\int x^n dx = \frac{x^{n+1}}{n+1} + C$$. Here, n = 1.3, so we add 1 to the exponent and divide by the new exponent.

Step 3: Integrate 7x^2.5

\[\int 7x^{2.5} dx = 7 \cdot \frac{x^{3.5}}{3.5} + C = 2x^{3.5} + C\]
We first pull out the constant 7, then use the power rule. The coefficient simplifies to 2 after combining the numbers.

Step 4: Combine results

\[\frac{x^{2.3}}{2.3} + 2x^{3.5} + C\]
We add the results from steps 2 and 3, keeping a single constant of integration C.

Problem 2

\[\int \frac{x + \sqrt{x}}{x} dx\]

Step 1: Simplify the integrand

\[\int (1 + x^{-1/2}) dx\]
We simplify the fraction by dividing each term by x. $$\sqrt{x} = x^{1/2}$$, so $$\frac{\sqrt{x}}{x} = x^{-1/2}$$.

Step 2: Integrate term by term

\[\int 1 dx + \int x^{-1/2} dx\]
We split the integral and integrate each term separately.

Step 3: Evaluate integrals

\[x + 2x^{1/2} + C\]
For the first term, $$\int 1 dx = x$$. For the second term, we use the power rule: $$\int x^{-1/2} dx = \frac{x^{1/2}}{1/2} = 2x^{1/2}$$.

Step 4: Simplify

\[x + 2\sqrt{x} + C\]
We write $$x^{1/2}$$ as $$\sqrt{x}$$ for a more familiar form.

Problem 3

\[\int \frac{1 + \cos^2\theta}{\cos^2\theta} d\theta\]

Step 1: Simplify the fraction

\[\int (1 + \sec^2\theta) d\theta\]
We recognize that $$\frac{1}{\cos^2\theta} = \sec^2\theta$$ and rewrite the expression accordingly.

Step 2: Integrate term by term

\[\int 1 d\theta + \int \sec^2\theta d\theta\]
We split the integral and integrate each term separately.

Step 3: Evaluate integrals

\[\theta + \tan\theta + C\]
For the first term, $$\int 1 d\theta = \theta$$. For the second term, we use the fact that $$\int \sec^2\theta d\theta = \tan\theta + C$$.
